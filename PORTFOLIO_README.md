## ✨ Features

- 🍎 **Apple boot screen** — authentic logo splash on load
- 🏝️ **Dynamic Island** — tap for instant contact shortcuts
- ⏰ **Live clock** — real-time status bar with current time
- 🔔 **Lock screen notifications** — achievements displayed as iOS notifs
- 📱 **App grid** — 8 tappable app icons with spring animations
- 🚀 **5 full app screens** — About, Skills, Projects, Certs, Contact
- 📲 **iOS slide-up transitions** — native-feel app open/close
- 💬 **Contact form** — sends directly to WhatsApp
- 🌊 **Ambient background** — animated glowing orbs
- 📡 **Zero dependencies** — pure HTML/CSS/JS, single file

Visit the live site → **[your-portfolio.vercel.app](https://your-portfolio.vercel.app)**

```bash
# Clone the repo
git clone https://github.com/Project30Hub/portfolio.git

```

```
1. Fork this repo on GitHub
2. Go to vercel.com → Import your repo
3. Hit Deploy — live in 60 seconds
```

## 📸 Sections Overview

### 🏠 Home Screen
Lock screen with live time, notification cards showing achievements, profile widget with stats, app grid, and dock.

### 🏝️ Dynamic Island
Tap the Dynamic Island at the top to open a contact sheet with direct links to WhatsApp, Email, LinkedIn, and GitHub.

### 📱 App Navigation
Tap any icon → app slides up with iOS spring animation. Tap back → slides down and dismisses.
